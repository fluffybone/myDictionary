# myDictionary

Создавайте персональный словарь выученных слов. Проверяйте, насколько хорошо вы их запомнили — функция проверки автоматически подбирает случайные слова для тренировки.

> запуск локально
команда `docker compose up -d --build`
Фронт **http://localhost:5173/**
Бэкенд
> запуск на **http://localhost:8000/**
> docs **http://localhost:8000/docs**

> запуск прод сборки `docker compose -f docker-compose.prod.yml up -d --build`

## Установка сайта как приложения (PWA)

В проект добавлена поддержка PWA: `manifest`, `service worker`, мобильные иконки и режим `standalone`.

### Что важно для пользователей

Для установки на телефон сайт должен открываться по `https://` (это обязательное условие для PWA на реальном устройстве).
На Android в поддерживаемых браузерах на сайте появляется баннер с кнопкой `Установить`.

### Android (Chrome)

1. Открыть сайт в Chrome.
2. Нажать меню браузера (три точки).
3. Выбрать `Установить приложение` или `Добавить на главный экран`.
4. Подтвердить установку.

### iPhone (Safari)

В Safari системный баннер установки может не появляться, поэтому используйте ручной способ:

1. Открыть сайт в Safari.
2. Нажать кнопку `Поделиться`.
3. Выбрать `На экран "Домой"`.
4. Нажать `Добавить`.

### Что получает пользователь после установки

- Запуск как обычного приложения с иконки на домашнем экране.
- Отдельное окно без адресной строки браузера.
- Быстрый повторный запуск за счет кэширования статических файлов.
