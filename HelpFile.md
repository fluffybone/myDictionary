> Чтобы изменения в коде на вашем компьютере мгновенно появлялись внутри контейнера

Пробросить папку (Volume): Сказать Докеру: «Не копируй файлы один раз при сборке, а смотри прямо в мою папку на компьютере».

Включить авто-перезагрузку: Сказать серверу (uvicorn): «Если файлы изменились, перезагрузись».

Это делается при запуске контейнера. Пересобирать образ (docker build) не нужно

```
docker run -d -p 8000:8000 \
  -v "${PWD}:/app" \
  --name backend-dev \
  my-backend \
  uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

-v "${PWD}:/app": Это значит «Возьми текущую папку (${PWD}) на моем компьютере и вставь её вместо папки /app внутри контейнера». Теперь они как зеркало.

uvicorn ... --reload: Мы перезаписываем команду запуска, добавляя флаг --reload. Теперь uvicorn следит за изменениями файлов.
